bacds version 0.01
==================

scripts from the bacds.org server

INSTALLATION

To install this module and all the scripts type the following:

   perl Makefile.PL
   make
   make test
   make install

(You can also do "make diff-with-prod" before "make install" to see what the
changes will be.)
